# Outris Identity Tools

| Tool Name | Credits | Description |
|-----------|---------|-------------|
| **get_name** | 2 | **What it does:** identifying the owner name of a phone number.<br>**Input:** Phone number.<br>**Returns:** names and confidence scores. |
| **get_email** | 2 | **What it does:** Finds email addresses linked to a phone number. |
| **get_address** | 2 | **What it does:** Finds physical addresses associated with a phone number. |
| **get_identity_profile** | 3 | **What it does:** Comprehensive identity report. Fetches names, emails, addresses, metadata, and risk scores in one go. |
| **check_online_platforms** | 1 | **What it does:** Checks if a phone number is registered on major global platforms (Amazon, Instagram, Snapchat). |
| **check_digital_commerce_activity** | 1 | **What it does:** Checks if a phone number has been used for any digital commerce activity. |
| **check_breaches** | 1 | **What it does:** Checks if a phone number or email has been exposed in known data breaches. |
| **get_alternate_phones** | 2 | **What it does:** Get other phone numbers belonging to the same person. |
